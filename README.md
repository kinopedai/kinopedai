# my-kinoko

きのこ識別アプリケーション

## 起動手順

1. 依存関係をインストール
   ```bash
   npm install
   ```

2. 開発サーバーを起動
   ```bash
   npm run dev
   ```

3. ブラウザで http://localhost:30000 にアクセス

## その他のコマンド

- プロダクション用ビルド: `npm run build`
- プロダクションモードで起動: `npm run start`
- リンター実行: `npm run lint`
